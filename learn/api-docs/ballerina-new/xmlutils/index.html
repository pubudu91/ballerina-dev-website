Module : xmlutils

Version : 0.5.0

Module overview

This module provides utility functions to manipulate the built-in xml data type. It provides APIs to convert a json to an xml or convert a table to an xml.

Records

JsonOptions

Represents a record type to provide configurations for the JSON to XML conversion.

Functions

fromJSON

Converts a JSON object to an XML representation.

fromTable

Converts a table to its XML representation.
